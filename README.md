# Super Mini Mario

A small browser platformer in a single HTML file — no build step, no dependencies.

## Play

Open `mario.html` in any modern browser (double-click it, or run `open mario.html` on macOS).

## Controls

- **← / →** — move
- **↑ / Space** — jump (press again in the air to **double jump**)
- **Enter** — advance menus / start a level
- **R** — restart the game

## Features

- **10 levels**, easiest to hardest, each framed by a start and end **castle**.
- **Mystery `?` boxes** — bonk from below for coins, a size increase, a speed boost, or a monster. Floating signs show what you won.
- **Two monster types** — stomp the brown ones once; the green armored ones take **two** stomps.
- **Hazards** — deadly **spikes** and spinning **saw blades** (horizontal and vertical), plus bottomless pits.
- Lives, score, and little WebAudio sound effects for jumps, coins, stomps, and entering/leaving castles.
